Apa itu Profesi Software Engineer ?

Insinyur perangkat lunak atau lebih sering dikenal dengan Software Engineer adalah seorang profesional yang tugasnya merancang, merealisasikan, mengembangkan, memelihara, menguji, dan mengevaluasi perangkat lunak komputer (software).

Tugas-Tugas Software Engineer

  1. Mendesain dan Merawat sistem software
  2. Testing dan Evaluasi program software baru
  3. Mengoptimasi speed dan scalability software
  4. Menulis dan menguji kode pemrograman
  5. Memberikan konsultasi ke client, sesama engineer, specialist computer security, dan stakeholder lainnya

Front End Development Pengembangan kode yang menciptakan element visual dari perangkat lunak, aplikasi atau situs web. Bahasa front End : HTML => Struktur CSS => mempercantik warna, bentuk Java Script => Interaksi

Software Development Life Cycle (SDLC)

  1. Planning & Analisys
  2. Design => Rancangan/Sketsa
  3. Implementasi (coding) => Software Enginner
  4. Testing & Integration => test system dan integrasi FE, BE,
  5. Deployment
  6. Maintenance

Version Control System and Repository

Version Control System => system yang mencatat setiap perubahan sebuah atau kumpulan file (code).

Source Code Repository => Tempat penyimpanan file projek yang berupa file source code, file gambar, text dan file lainnya.
Contoh: Github, Gitlab, Bitbucket

Developer Tools => alat yang harus dipersiapkan :

  1. Visual Studio Code => Code Editor
  2. Google Chrome => menampilkan hasil coding
  3. Git => penyimpanan file coding
